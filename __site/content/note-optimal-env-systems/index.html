A note on "Optimal Design of Controlled Environment Agricultural Systems (...)"

Posted 2021-03-17

This is just a basic note on Cetegen and Stuber's paper (apologies for the paywall) published a few days ago, Optimal Design of Controlled Environment Agricultural Systems Under Market Uncertainty. This post "simplifies" problem (4) from a bilevel optimization problem to a single (convex) optimization problem which can be readily solved. As a side note, I only skimmed the references provided, and some were too complicated for me to understand, especially at first glance.[1] It is possible this technique is well known in the specific literature referenced, but hidden under mountains of notation and definitions. I also note that this is not the problem they (approximately) solve, but the method presented here might be useful in that case, too.

The One (Duality) Trick Doctors Don't Want You To Know™

I've used this trick before in a few other papers, with the main example being a paper coauthored with Kunal Shah and Mac Schwager, found here, specifically in equation (8) and below, starting on page 10.

The basic (very general!) idea is to replace a "min-max" optimization problem with a "min" optimization problem. For example, say we are given the following optimization problem

minimizemax⁡g(y)≤0f(x,y)subject toh(x)≤0, \begin{aligned} & \text{minimize} && \max_{g(y) \le 0} f(x, y)\\ & \text{subject to} && h(x) \le 0, \end{aligned} ​minimizesubject to​​g(y)≤0max​f(x,y)h(x)≤0,​

with variables x∈Rmx \in \mathbb{R}^mx∈Rm and y∈Rny \in \mathbb{R}^ny∈Rn and functions f:Rm×Rn→Rf : \mathbb{R}^m \times \mathbb{R}^n \to \mathbb{R}f:Rm×Rn→R, g:Rn→Rg:\mathbb{R}^n \to \mathbb{R}g:Rn→R, and h:Rm→Rh: \mathbb{R}^m \to \mathbb{R}h:Rm→R. Now, consider the usual Lagrangian of the "inner problem" (the one with the max over yyy), which we know is

L(x,y,λ)=f(x,y)−λg(y). L(x, y,\lambda) = f(x, y) - \lambda g(y). L(x,y,λ)=f(x,y)−λg(y).

If we define

fˉ(x,λ)=sup⁡yL(x,y,λ), \bar f(x, \lambda) = \sup_{y} L(x, y, \lambda), fˉ​(x,λ)=ysup​L(x,y,λ),

then, for any λ≥0\lambda \ge 0λ≥0 and any feasible yyy (i.e., yyy that satisfies g(y)≤0g(y) \le 0g(y)≤0), we have that

fˉ(x,λ)≥L(x,y,λ)=f(x,y)−λg(y)≥f(x,y). \bar f(x, \lambda) \ge L(x, y, \lambda) = f(x, y) - \lambda g(y) \ge f(x, y). fˉ​(x,λ)≥L(x,y,λ)=f(x,y)−λg(y)≥f(x,y).

(The first inequality follows from the definition of sup⁡\supsup while the last inequality follows since λ≥0\lambda \ge 0λ≥0 and g(y)≤0g(y) \le 0g(y)≤0 which means that λg(y)≤0\lambda g(y) \le 0λg(y)≤0.) So, for any xxx, we know that, if you give me any λ≥0\lambda \ge 0λ≥0, then fˉ(x,λ)\bar f(x, \lambda)fˉ​(x,λ) is an "overestimator" of f(x,y)f(x, y)f(x,y) for any feasible yyy.

But, since this is true for any λ≥0\lambda \ge 0λ≥0 and xxx, then certainly

inf⁡λ≥0f(x,λ)≥sup⁡g(y)≤0f(x,y), \inf_{\lambda \ge 0} f(x, \lambda) \ge \sup_{g(y) \le 0} f(x, y), λ≥0inf​f(x,λ)≥g(y)≤0sup​f(x,y),

for any xxx.[2]

If we use our new overestimator fˉ\bar ffˉ​ instead of fff, our new problem is now a simple optimization problem

minimizefˉ(x,λ)subject toh(x)≤0λ≥0, \begin{aligned} & \text{minimize} && \bar f(x, \lambda)\\ & \text{subject to} && h(x) \le 0\\ &&& \lambda \ge 0, \end{aligned} ​minimizesubject to​​fˉ​(x,λ)h(x)≤0λ≥0,​

that is not in min-max form and requires no other special techniques to solve. The optimal value of this problem need not be the same as that of the original, but is always guaranteed to be at least as large.

Of course, this can help, but it certainly doesn't solve our problem. We just need one more piece to the puzzle!

The hammer

If you've studied a bit of convex analysis the punchline is this: the inequality we have for fff and fˉ\bar ffˉ​ above holds exactly at equality when fff is concave in yyy and ggg is convex in yyy. More specifically

inf⁡λ≥0fˉ(x,λ)=sup⁡g(y)≤0f(x,y), \inf_{\lambda \ge 0} \bar f(x, \lambda) = \sup_{g(y) \le 0} f(x, y), λ≥0inf​fˉ​(x,λ)=g(y)≤0sup​f(x,y),

for any xxx.[3]

When this is true, the new problem has the same optimal value as the original and any solution xxx for the original is a solution to the new problem! (Why?)

I won't cover more since the specifics don't matter too much, but the general idea is simple enough, and we now have all the parts to convert the min-max problem (4) of Cetegen and Stuber to a simple convex optimization problem.

The (new and shiny) problem

Things are mostly algebra from here on out, so apologies in advance, I guess. I will leave much of the "hard" work to the reader :)

The complete problem (4) is, as written in the paper, using (mostly) their notation:

minimizemax⁡M∈MxTMxsubject torTx≥rmin1Tx=10≤x≤1, \begin{aligned} & \text{minimize} && \max_{M \in \mathcal{M}} x^TMx \\ & \text{subject to} && r^Tx \ge r_\mathrm{min}\\ &&& 1^Tx = 1\\ &&& 0 \le x \le 1, \end{aligned} ​minimizesubject to​​M∈Mmax​xTMxrTx≥rmin​1Tx=10≤x≤1,​

with variable x∈Rnx \in \mathbb{R}^nx∈Rn, where rrr is some vector of returns (but the specifics don't matter) and M\mathcal{M}M is:

M={M≥0∣Mij−≤Mij≤Mij+,  i,j=1,…,n}. \mathcal{M} = \{M \ge 0 \mid M^-_{ij} \le M_{ij} \le M^+_{ij}, ~~ i, j=1, \dots, n\}. M={M≥0∣Mij−​≤Mij​≤Mij+​,  i,j=1,…,n}.

In other words, M\mathcal{M}M is the set of positive semidefinite matrices (M≥0M \ge 0M≥0) whose entries lie between those of M−M^-M− and M+M^+M+. I've also dropped some constant terms in the objective since those don't change the problem.

In this case, the "inner" optimization problem is the one in the objective, which is just

maximizexTMxsubject toMij−≤Mij≤Mij+,  i,j=1,…,nM≥0, \begin{aligned} & \text{maximize} && x^TMx \\ & \text{subject to} && M^-_{ij} \le M_{ij} \le M^+_{ij}, ~~ i, j=1, \dots, n\\ &&& M \ge 0, \end{aligned} ​maximizesubject to​​xTMxMij−​≤Mij​≤Mij+​,  i,j=1,…,nM≥0,​

with variable M∈Rn×nM \in \mathbb{R}^{n\times n}M∈Rn×n. We can easily write a (slightly not canonical) Lagrangian:

L(x,M,Λ+,Λ−)=xTMx−tr(Λ+(M−M+))+tr(Λ−(M−M−)), L(x, M, \Lambda^+, \Lambda^-) = x^TMx - \mathrm{tr}(\Lambda^+(M - M^+)) + \mathrm{tr}(\Lambda^-(M - M^-)), L(x,M,Λ+,Λ−)=xTMx−tr(Λ+(M−M+))+tr(Λ−(M−M−)),

where Λ+,Λ−∈R+n×n\Lambda^+, \Lambda^- \in \mathbb{R}^{n\times n}_+Λ+,Λ−∈R+n×n​ are elementwise nonnegative. (The Lagrangian is non-canonical because I have not included the constraint M≥0M \ge 0M≥0, which we will enforce below.) It is not hard to show that

sup⁡M≥0L(x,M,Λ+,Λ−)={tr(Λ+M+)−tr(Λ−M−)xxT≤Λ+−Λ−+∞otherwise. \sup_{M \ge 0} L(x, M, \Lambda^+, \Lambda^-) = \begin{cases} \mathrm{tr}(\Lambda^+M^+) - \mathrm{tr}(\Lambda^-M^-) & xx^T \le \Lambda^+ - \Lambda^-\\ + \infty & \text{otherwise}. \end{cases} M≥0sup​L(x,M,Λ+,Λ−)={tr(Λ+M+)−tr(Λ−M−)+∞​xxT≤Λ+−Λ−otherwise.​

As before, the inequality between matrices is with respect to the semidefinite cone.

Plugging this back into the original problem formulation, we now have a convex optimization problem:

minimizetr(Λ+M+)−tr(Λ−M−)subject torTx≥rminxxT≤Λ+−Λ−1Tx=10≤x≤1Λij+,Λij−≥0,i,j=1,…,n. \begin{aligned} & \text{minimize} && \mathrm{tr}(\Lambda^+M^+) - \mathrm{tr}(\Lambda^-M^-)\\ & \text{subject to} && r^Tx \ge r_\mathrm{min}\\ &&& xx^T \le \Lambda^+ - \Lambda^-\\ &&& 1^Tx = 1\\ &&& 0 \le x \le 1\\ &&& \Lambda^+_{ij}, \Lambda^-_{ij} \ge 0, \quad i,j =1, \dots, n. \end{aligned} ​minimizesubject to​​tr(Λ+M+)−tr(Λ−M−)rTx≥rmin​xxT≤Λ+−Λ−1Tx=10≤x≤1Λij+​,Λij−​≥0,i,j=1,…,n.​

The (extra!) variables Λ+,Λ−∈Rn×n\Lambda^+, \Lambda^- \in \mathbb{R}^{n\times n}Λ+,Λ−∈Rn×n are included along with the original variable x∈Rnx \in \mathbb{R}^nx∈Rn, and the same problem data as before. This problem, by use of the Schur complement applied to the semidefinite inequality, is easily recast into standard SDP form and can be solved by most standard convex optimization problem solvers, such as SCS or Mosek.
